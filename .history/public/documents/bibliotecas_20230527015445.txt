

Tecnologias, ferramentas e bibliotecas Utilizadas no projeto

1. Express: Um framework web para Node.js que simplifica o desenvolvimento de aplicativos web.
11. javascript: Uma linguagem de programação usada para adicionar interatividade e lógica a páginas web.
2. Node.js: Uma plataforma de tempo de execução de JavaScript baseada em eventos que permite executar código JavaScript no lado do servidor.
9. Bootstrap: Um framework CSS popular que fornece componentes e estilos predefinidos para ajudar no desenvolvimento responsivo e ágil de páginas web.
7. MongoDB: Um banco de dados NoSQL baseado em documentos, que pode ser usado com o Node.js para armazenar e recuperar dados.
8. Mongoose: Uma biblioteca de modelagem de objetos do MongoDB para Node.js, que simplifica a interação com o banco de dados MongoDB.
5. Mustache: Uma linguagem de modelo que permite a separação de dados e apresentação em aplicativos web.
6. Git: Um sistema de controle de versão distribuído usado para rastrear e gerenciar o código-fonte do projets
10. Github: Uma plataforma de hospedagem de código-fonte que usa o Git para controle de versão.
